# 如何访问 Phase 5 测试屏幕

## 方法 1: 通过设置页面（推荐）

1. **打开应用**
2. **进入设置页面**
   - 从主屏幕或任何页面导航到"设置"（Settings）
3. **找到"开发工具"部分**
   - 滚动到底部
   - 找到"开发工具"（Developer Tools）部分
4. **点击测试按钮**
   - 点击 "🧪 Phase 5 测试 (Craving 集成)" 按钮
   - 即可进入测试屏幕

## 方法 2: 使用深度链接（开发环境）

### 在 Expo Go 中：
1. **打开 Expo Go 应用**
2. **在浏览器或终端中使用深度链接**
   ```
   kitchenflow://dev/test-phase5
   ```

### 在开发环境中：
如果您在开发服务器中，可以在浏览器地址栏输入：
```
exp://localhost:8081/--/dev/test-phase5
```

## 方法 3: 直接修改代码导航

如果您想在代码中直接导航，可以在任何组件中添加：

```typescript
navigation.navigate('Phase5Test');
```

## 方法 4: 在 HomeScreen 添加快捷入口（可选）

如果您想更方便地访问，可以在 HomeScreen 或 QuickAccessBar 中添加一个测试按钮。

## 测试屏幕功能

进入测试屏幕后，您可以：

1. **运行自动化测试**
   - 点击 "Run Phase 5 Tests" 按钮
   - 查看测试结果摘要
   - 查看详细的测试输出

2. **查看测试说明**
   - 屏幕底部有手动测试指南
   - 参考 `docs/PHASE5_TESTING_GUIDE.md` 获取详细步骤

## 快速访问路径

```
应用启动 → 设置 (Settings) → 开发工具 → Phase 5 测试
```

## 注意事项

- 测试屏幕仅在开发环境中使用
- 生产版本中应移除或隐藏这些测试入口
- 确保应用已正确配置路由（已在 App.tsx 中添加）
